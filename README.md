# OpenSUSE-debloat-script

## v0.1 — Functional, work in progress
A post-install debloat script for **openSUSE Tumbleweed** (will probably work on **Leap** also). Removes games, unnecessary office apps, multimedia cruft, welcome/tour screens, unused YaST modules, and optional app stores. Keeps your system fully functional (hopefully).

> Tested on openSUSE Tumbleweed. Run after a fresh install.
